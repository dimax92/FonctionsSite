Comment vendre son pantalon

Une fois connecte a votre compte.

Pour vendre votre pantalon sur machatvente.com il faut d'abord cliquer sur "Déposer une annonce".

Dans le champ "Nom du produit" vous insérer le nom du modèle de pantalon que vous possédez.

Dans le champ "Marque du produit" vous insérer la marque de votre pantalon.

Cliquez sur "Choisir une vidéo" et ensuite choisissez la vidéo de présentation de votre pantalon, nous vous conseillons de prendre votre video en plaine lumière pour que le client potentiel vois bien en détail votre pantalon et de montrer le pantalon sous tous les angles possibles pour que le client potentiel est une vue détaillé de votre pantalon. Filmez aussi l'interieur du pantalon si vous pouvez pour que l'acheteur potentiel vois qu'il n'y a pas de problème particulier. Le poids de la video ne doit pas depasser 100mb, la video doit être courte.

Dans le champ "Prix du produit" vous insérer le prix que vous voulez pour votre pantalon.

Dans le champ "Devise" vous insérer la devise que vous voulez(euros, dollars, bitcoin, …).

Dans le champ "Description du produit" vous insérer une présentation générale de votre pantalon et aussi informer sur l'entretien du pantalon.

Pour donner des détails sur votre pantalon cliquer sur le "+" a droite de "Details du produit".
Les détails que je vous conseille d'indiquer sont en vous donnant les caracteristiques a titre d'exemple pour le pantalon DOCKERS ALPHA ORIGINAL - Chino(pantalon Alpha original):
-Composition(par exemple vous pouvez mettre si vous voulez: 98% coton, 2% élasthanne)
-Conseils d'entretien(par exemple vous pouvez mettre si vous voulez: Lavage en machine à 30°C)
-Taille(par exemple vous pouvez mettre si vous voulez: Normale)
-Fermeture(par exemple vous pouvez mettre si vous voulez: Braguette avec fermeture éclair dissimulée)
-Poches(par exemple vous pouvez mettre si vous voulez: Poches arrière, poches latérales)
-Motif / Couleur(par exemple vous pouvez mettre si vous voulez: Couleur unie)
-Référence(par exemple vous pouvez mettre si vous voulez: DO822E00J-B12)
-Taille du mannequin(par exemple vous pouvez mettre si vous voulez: Notre mannequin mesure 188 cm et porte une taille 32x32)
-Coupe(par exemple vous pouvez mettre si vous voulez: Tapered)
-Longueur(par exemple vous pouvez mettre si vous voulez: Longue)
-Longueur intérieure de jambe(par exemple vous pouvez mettre si vous voulez: 81 cm en taille 32x32)
-Longueur extérieure de jambe(par exemple vous pouvez mettre si vous voulez: 105 cm en taille 32x32)
Dans le champ "Quantité" vous insérer le nombre de pantalons du meme modèle que vous souhaitez vendre.

Dans le champ "Type de produit" vous insérer "pantalon".

Dans le champ "Condition" vous insérer "neuf" si elle est neuf sinon insérer "occasion".

Dans le champ "Coordonnées de contact" vous insérer vos coordonnées par lesquelles le client peut vous contacter.

Dans le champ "Coordonnées du produit" vous insérer le lieu exact ou le client peut récupérer le pantalon.

L'exemple que nous prenons est un pantalon en chino de type Dockers, mais il existe beaucoup d'autres matieres comme le Cachemire, Cachemire (100%), Coton, Coton (100%), Crocheté, Cuir, Côtelé, Denim, Dentelle, Flanelle chaude, Hardshell, Imitation cuir, Jersey, Laine, Laine (100%), Lin, Lin (100%), Lyocell (100%), Mesh, Paillettes, Polaire, Polyester, Satin, Softshell, Soie, Strié, Sweat, Textile, Tressé, Tricots, Viscose et Viscose (100%).
