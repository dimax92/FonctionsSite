Comment vendre sa playstation

Une fois connecte a votre compte.

Pour vendre votre playstation sur machatvente.com il faut d'abord cliquer sur "Déposer une annonce".

Dans le champ "Nom du produit" vous insérer le nom du modèle de playstation que vous possédez.

Dans le champ "Marque du produit" vous insérer la marque de votre playstation a savoir "Sony".

Cliquez sur "Choisir une vidéo" et ensuite choisissez la vidéo de présentation de votre playstation, nous vous conseillons de prendre votre video en plaine lumière pour que le client potentiel vois bien en détail votre playstation et de montrer la playstation sous tous les angles possibles pour que le client potentiel est une vue détaillé de votre playstation. Montrez qu'elle marche bien en faisant une démonstration de l'utilisation de votre playstation et filmez aussi l'interieur de la playstation si vous pouvez pour que l'utilisateur potentiel vois qu'il n'y a pas de problème particulier. Le poids de la video ne doit pas depasser 100mb, la video doit être courte.

Dans le champ "Prix du produit" vous insérer le prix que vous voulez pour votre playstation.

Dans le champ "Devise" vous insérer la devise que vous voulez(euros, dollars, bitcoin, …).

Dans le champ "Description du produit" vous insérer une présentation générale de votre playstation.

Pour donner des détails sur votre playstation cliquer sur le "+" a droite de "Details du produit".
Les détails que je vous conseille d'indiquer sont en vous donnant les caracteristiques a titre d'exemple pour la playstation 5(la derniere qui vient de sortir):
-Mémoire extensible(par exemple vous pouvez mettre: Oui)
-Mémoire interne(par exemple vous pouvez mettre: 825 Go)
-Type d'extension de mémoire(par exemple vous pouvez mettre: SSD)
-Support(par exemple vous pouvez mettre: Blu-Ray 4K)
-RAM(par exemple vous pouvez mettre: 16 Go)
-Architecture graphique(par exemple vous pouvez mettre: AMD RDNA 2)
-Finesse de gravure(par exemple vous pouvez mettre: 7 nm)
-Compatible Surround(par exemple vous pouvez mettre: Tempest Engine)
-HDR(par exemple vous pouvez mettre: HDR10)
-Définition maximale supportée(par exemple vous pouvez mettre: 8K)
-Framerate Maximal(par exemple vous pouvez mettre: 120 Hz)
-Norme Wi-Fi(par exemple vous pouvez mettre: Wi-Fi 6 (ax))
-Bluetooth(par exemple vous pouvez mettre: Oui)
-Nombre de port HDMI(par exemple vous pouvez mettre: 1)
-Nombre de ports USB(par exemple vous pouvez mettre: 2)
-Couleur(par exemple vous pouvez mettre: Blanc)
-Apparence(par exemple vous pouvez mettre: Plastique)
-Date de sortie(par exemple vous pouvez mettre: 19/11/2020)
-Contrôleur(par exemple vous pouvez mettre: Dualsense)
-Compatible Surround(par exemple vous pouvez mettre: Tempest Engine)

Dans le champ "Quantité" vous insérer le nombre de playstation du meme modèle que vous souhaitez vendre.

Dans le champ "Type de produit" vous insérer "console".

Dans le champ "Condition" vous insérer "neuf" si elle est neuf sinon insérer "occasion".

Dans le champ "Coordonnées de contact" vous insérer vos coordonnées par lesquelles le client peut vous contacter.

Dans le champ "Coordonnées du produit" vous insérer le lieu exact ou le client peut récupérer la playstation.

Prenez bien soin de nettoyer votre playstation si elle est sale.

Si vous le voulez les caracteristiques peuvent s'appliquer aussi bien a une Xbox.
