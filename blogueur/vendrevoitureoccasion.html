Comment vendre une voiture d'occasion

Pour vendre une voiture sur machatvente.com il faut d'abord cliquer sur "Déposer une annonce".

Dans le champ "Nom du produit" vous insérer le nom du modèle de véhicule que vous possédez.

Dans le champ "Marque du produit" vous insérer la marque de votre véhicule.

Cliquez sur "Choisir une vidéo" et ensuite choisissez la vidéo de présentation de votre véhicule.

Dans le champ "Prix du produit" vous insérer le prix que vous voulez pour le véhicule.

Dans le champ "Devise" vous insérer la devise que vous voulez(euros, dollars, bitcoin, …).

Dans le champ "Description du produit" vous insérer une présentation générale de votre véhicule.

Pour donner des détails techniques sur votre véhicule cliquer sur le "+" a droite de "Details du produit".
Les détails que je vous conseille d'indiquer sont:
-Nombre de places
-Volume de coffre
-Empattement
-Poids à vide
-Réservoir
-Consommation urbaine
-Consommation mixte
-Consommation extra-urbaine
-Nombre de cylindres
-Nombre de soupapes par cylindre
-Cylindrée
-Puissance din
-Couple moteur
-Puissance fiscale
-Position du moteur
-Alimentation
-Suralimentation/type
-Vitesse maximum
-Accéleration 0/100km/h
-Transmission
-Boite
-Nb. vitesses
-Distribution
-Position du moteur
-Direction assistée
-Carrosserie
-Diamètre braquage trottoirs
-Diamètre braquage murs
-Suspension avant
-Suspension arrière
-Freins
-Largeur pneu avant
-Largeur pneu arrière
-Rapport h/L pneu avant
-Rapport h/L pneu arrière
-Diamètre des jantes avant
-Diamètre des jantes arrière
-Intervalle entretien
-Garantie mois
-Nationalité du constructeur
-Début commercialisation
-Fin commercialisation

Dans le champ "Quantité" vous insérer le nombre de véhicules que vous souhaitez vendre.

Dans le champ "Type de produit" vous insérer "voiture".

Dans le champ "Condition" vous insérer "occasion".

Dans le champ "Coordonnées de contact" vous insérer vos coordonnées par lesquelles le client peut vous contacter.

Dans le champ "Coordonnées du produit" vous insérer le lieu exact ou se trouve le véhicule.
