Comment vendre son casque audio

Une fois connecte a votre compte.

Pour vendre son casque audio sur machatvente.com il faut d'abord cliquer sur "Déposer une annonce".

Dans le champ "Nom du produit" vous insérer le nom du modèle du casque audio que vous possédez.

Dans le champ "Marque du produit" vous insérer la marque de votre casque audio.

Cliquez sur "Choisir une vidéo" et ensuite choisissez la vidéo de présentation de votre casque audio, nous vous conseillons de prendre votre video en pleine lumière pour que le client potentiel vois bien en détail votre casque audio et de montrer le casque audio sous tous les angles possibles pour que le client potentiel est une vue détaillé de votre casque audio. Montrez qu 'il marche bien en faisant une démonstration de l'utilisation de votre casque audio. Le poids de la video ne doit pas depasser 100mb, la video doit être courte.

Dans le champ "Prix du produit" vous insérer le prix que vous voulez pour votre casque audio.

Dans le champ "Devise" vous insérer la devise que vous voulez(euros, dollars, bitcoin, …).

Dans le champ "Description du produit" vous insérer une présentation générale de votre casque audio.

Pour donner des détails sur votre casque audio cliquer sur le "+" a droite de "Details du produit".
Les détails que je vous conseille d'indiquer sont:
-Famille de casques(par exemple vous pouvez mettre: Casque Bluetooth)
-Modèle(par exemple vous pouvez mettre: Serre tête)
-Type de casque(par exemple vous pouvez mettre: Fermé)
-Restitution du son(par exemple vous pouvez mettre: Stéréo)
-Type de coussinets(par exemple vous pouvez mettre: Supra-aural)
-Micro intégré au casque(par exemple vous pouvez mettre: Oui)
-Système actif de réduction du bruit(par exemple vous pouvez mettre: Non)
-Type d'interface audio(par exemple vous pouvez mettre: Adaptateur Jack 3,5 mm, Bluetooth)
-Type de microphone(par exemple vous pouvez mettre: Fixe)
-Poids (avec cordon et piles)(par exemple vous pouvez mettre: 210 g)
-Pliant(par exemple vous pouvez mettre: Oui)
-Longueur de câble totale(par exemple vous pouvez mettre: 1.3 m)
-Câble rallonge(par exemple vous pouvez mettre: Non)
-Housse ou boîtier de transport(par exemple vous pouvez mettre: Oui)
-Nombre de câbles fournis(par exemple vous pouvez mettre: 2)
-Autres accessoires fournis(par exemple vous pouvez mettre: Housse de transport. câble audio. câble micro USB. mousqueton)
-Commandes sur le câble(par exemple vous pouvez mettre: Volume + marche / arrêt)
-Commandes sur le casque(par exemple vous pouvez mettre: Volume + marche / arrêt)
-Câble amovible(par exemple vous pouvez mettre: Oui)
-Type d'alimentation(par exemple vous pouvez mettre: Batterie(s), Lithium Ion)
-Autonomie annoncée(par exemple vous pouvez mettre: 1 jr 16 h)
-Type de transmission sans fil(par exemple vous pouvez mettre: Bluetooth)
-Portée de la liaison sans fil(par exemple vous pouvez mettre: 10 m)

Dans le champ "Quantité" vous insérer le nombre de casques audios du meme modèle que vous souhaitez vendre.

Dans le champ "Type de produit" vous insérer "casque audio".

Dans le champ "Condition" vous insérer "neuf" si il est neuf sinon insérer "occasion".

Dans le champ "Coordonnées de contact" vous insérer vos coordonnées par lesquelles le client peut vous contacter.

Dans le champ "Coordonnées du produit" vous insérer le lieu exact ou le client peut récupérer le casque audio.

Prenez bien soin de nettoyer votre casque audio si il est sale.
