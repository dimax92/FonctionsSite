Site d'annonces produits par videos

Vous voulez vendre vos produits ou au contraire acheter de nouveaux produits. Ce site est fait pour vous. Il vous permet de deposer une annonce en ligne tres rapidement et simplement. Si vous souhaitez deposer une annonce il suffit de cliquer sur "Deposer une annonce" dans la barre de navigation". Si vous souhaitez consultez les diverses annonces il vous suffit de cliquer sur "Rechercher".

Comment deposer une annonce

S'inscrire

Pour pouvoir deposer une annonce sur le site il faut d'abord s'inscrire. Pour pouvoir remplir le formulaire d'inscription il faut cliquer sur "S'inscrire" sur la barre de navigation, ensuite il vous suffira de remplir le formulaire en fournissant votre email (verification d'identite) et un mot de passe securise.

S'authentifier

Si vous etes deja inscrit sur le site, il faudra vous authentifier. Pour vous authentifie il faut d'abord cliquer sur "S'authentifie", ensuite il faut remplir le formulaire avec votre adresse mail et votre mot de passe.

Deposer une annonce

Si vous vous etes authentifiee avec succes, vous pouvez acceder au formulaire pour deposer votre annonce en clquant sur "Deposer une annonce" dans la navigation. Une fois sur la page vous devrez remplir le formulaire. Une fois le formulaire rempli, cliquez sur Mettre en ligne.

Rechercher une annonce

Si vous souhaitez rechercher l'annonce qui vous convient, cliquez sur "Rechercher" dans la barre de navigation. Une fois sur la page, taper votre recherche dans la barre de recherche et ensuite cliquez sur le bouton "Rechercher". Les produits correspondants a votre recherche s'afficheront, si vous souhaitez avoir plus de details sur un produit en particulier cliquer dessus et vous serez rediriger vers la fiche complete du produit.
