Comment vendre son smartphone

Une fois connecte a votre compte.

Pour vendre son smartphone sur machatvente.com il faut d'abord cliquer sur "Déposer une annonce".

Dans le champ "Nom du produit" vous insérer le nom du modèle du smartphone que vous possédez.

Dans le champ "Marque du produit" vous insérer la marque de votre smartphone.

Cliquez sur "Choisir une vidéo" et ensuite choisissez la vidéo de présentation de votre smartphone, nous vous conseillons de prendre votre video en plaine lumière pour que le client potentiel vois bien en détail votre smartphone et de montrer le smartphone sous tous les angles possibles pour que le client potentiel est une vue détaillé de votre smartphone. Montrez qu 'il marche bien en faisant une démonstration de l'utilisation de votre smartphone et filmez aussi le dos de l'iphone sans la coque pour que l'utilisateur potentiel vois qu'il n'y a pas de problème particulier. Le poids de la video ne doit pas depasser 100mb, la video doit être courte.

Dans le champ "Prix du produit" vous insérer le prix que vous voulez pour votre smartphone.

Dans le champ "Devise" vous insérer la devise que vous voulez(euros, dollars, bitcoin, …).

Dans le champ "Description du produit" vous insérer une présentation générale de votre smartphone.

Pour donner des détails sur votre smartphone cliquer sur le "+" a droite de "Details du produit".

Dans le champ "Quantité" vous insérer le nombre de smartphones du meme modèle que vous souhaitez vendre.

Dans le champ "Type de produit" vous insérer "téléphone portable".

Dans le champ "Condition" vous insérer "neuf" si il est neuf sinon insérer "occasion".

Dans le champ "Coordonnées de contact" vous insérer vos coordonnées par lesquelles le client peut vous contacter.

Dans le champ "Coordonnées du produit" vous insérer le lieu exact ou le client peut récupérer le smartphone.

Prenez bien soin de nettoyer votre smartphone si il est sale.
