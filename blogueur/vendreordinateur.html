Comment vendre son ordinateur

Une fois connecte a votre compte.

Pour vendre son ordinateur sur machatvente.com il faut d'abord cliquer sur "Déposer une annonce".

Dans le champ "Nom du produit" vous insérer le nom du modèle d'ordinateur que vous possédez.

Dans le champ "Marque du produit" vous insérer la marque de votre ordinateur.

Cliquez sur "Choisir une vidéo" et ensuite choisissez la vidéo de présentation de votre ordinateur, nous vous conseillons de prendre votre video en plaine lumière pour que le client potentiel vois bien en détail votre ordinateur et de montrer l'ordinateur sous tous les angles possibles pour que le client potentiel est une vue détaillé de votre ordinateur. Montrez qu'il marche bien en faisant une démonstration de l'utilisation de votre ordinateur et filmez aussi l'interieur de l'ordinateur si vous pouvez pour que l'utilisateur potentiel vois qu'il n'y a pas de problème particulier. Le poids de la video ne doit pas depasser 100mb, la video doit être courte.

Dans le champ "Prix du produit" vous insérer le prix que vous voulez pour votre ordinateur.

Dans le champ "Devise" vous insérer la devise que vous voulez(euros, dollars, bitcoin, …).

Dans le champ "Description du produit" vous insérer une présentation générale de votre ordinateur.

Pour donner des détails sur votre ordinateur cliquer sur le "+" a droite de "Details du produit".
Les détails que je vous conseille d'indiquer sont:
-Processeur(exemple: Intel Core M-5Y31)
-Fréquence processeur(exemple: 1,1 GHz)
-Mémoire vive (RAM)(exemple: 8 Go)
-Type de mémoire vive(exemple: DDR3-SDRAM)
-Capacité de stockage principal(exemple: 256 Go)
-Type de stockage principal(exemple: SSD)
-Interface stockage principal(exemple: PCI-Express)
-Seconde baie disque(exemple: Non)
-Capacité de stockage secondaire(exemple: 0 Go)
-Calcul de la capacité totale (en Go)(exemple: 256 Go)
-Résolution WebCam(exemple: 0,9 Mpx)
-Pavé numérique(exemple: Non)
-Clavier retroéclairé(exemple: Oui)
-Taille d'écran(exemple: 12 ")
-Format de l'écran(exemple: 16:9)
-Tactile(exemple: Non)
-Définition de l'écran(exemple: 2304 x 1440)
-Aspect de l'écran(exemple: Brillant)
-Puce graphique(exemple: Intel HD Graphics 5300)
-Quantité de mémoire graphique(exemple: Partagée)
-SD / MMC(exemple: Non)
-MicroSD(exemple: Non)
-Nombre de ports USB 2.0(exemple: 0)
-Nombre de ports USB 3.0(exemple: 0)
-USB Type-C(exemple: 1)
-Ports Thunderbolt(exemple: 0)
-Support du NFC(exemple: Non)
-Système audio embarqué(exemple: Stéréo)
-Support du Wi-Fi(exemple: Wi-Fi 802.11ac)
-Support du Bluetooth(exemple: Bluetooth 4.0)
-Ethernet 10/100(exemple: Non)
-Ethernet 100/1000(exemple: Non)
-Sortie son(exemple: Stéréo)
-Sortie S/PDIF(exemple: Non)
-Sortie DisplayPort(exemple: Non)
-Sortie HDMI(exemple: Non)
-Système d'exploitation(exemple: Mac OS X 10.10 Yosemite)
-Format de l'alimentation(exemple: Prise)
-Autonomie annoncée(exemple: 10 h)
-Largeur(exemple: 28,05 cm)
-Hauteur(exemple: 1,31 cm)
-Profondeur(exemple: 19,65 cm)
-Poids (batterie incluse)(exemple: 0,92 kg)

Dans le champ "Quantité" vous insérer le nombre d'ordinateurs du meme modèle que vous souhaitez vendre.

Dans le champ "Type de produit" vous insérer "ordinateur".

Dans le champ "Condition" vous insérer "neuf" si il est neuf sinon insérer "occasion".

Dans le champ "Coordonnées de contact" vous insérer vos coordonnées par lesquelles le client peut vous contacter.

Dans le champ "Coordonnées du produit" vous insérer le lieu exact ou le client peut récupérer l'ordinateur.

Prenez bien soin de nettoyer votre ordinateur si il est sale.
