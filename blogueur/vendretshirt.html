Comment vendre son T-shirt

Une fois connecte a votre compte.

Pour vendre votre t-shirt sur machatvente.com il faut d'abord cliquer sur "Déposer une annonce".

Dans le champ "Nom du produit" vous insérer le nom du modèle de t-shirt que vous possédez.

Dans le champ "Marque du produit" vous insérer la marque de votre t-shirt.

Cliquez sur "Choisir une vidéo" et ensuite choisissez la vidéo de présentation de votre t-shirt, nous vous conseillons de prendre votre video en plaine lumière pour que le client potentiel vois bien en détail votre t-shirt et de montrer le t-shirt sous tous les angles possibles pour que le client potentiel est une vue détaillé de votre t-shirt. Filmez aussi l'interieur du t-shirt si vous pouvez pour que l'acheteur potentiel vois qu'il n'y a pas de problème particulier. Le poids de la video ne doit pas depasser 100mb, la video doit être courte.

Dans le champ "Prix du produit" vous insérer le prix que vous voulez pour votre t-shirt.

Dans le champ "Devise" vous insérer la devise que vous voulez(euros, dollars, bitcoin, …).

Dans le champ "Description du produit" vous insérer une présentation générale de votre t-shirt et aussi informer sur l'entretien du t-shirt.

Pour donner des détails sur votre t-shirt cliquer sur le "+" a droite de "Details du produit".
Les détails que je vous conseille d'indiquer sont en vous donnant les caracteristiques a titre d'exemple pour le t-shirt Nike Sportswear CLUB TEE - T-shirt basique:
-Composition(par exemple vous pouvez mettre si vous voulez: 100% coton)
-Matière(par exemple vous pouvez mettre si vous voulez: Jersey)
-Conseils d'entretien(par exemple vous pouvez mettre si vous voulez: Lavage en machine à 30°C)
-Forme du col(par exemple vous pouvez mettre si vous voulez: Col rond)
-Motif / Couleur(par exemple vous pouvez mettre si vous voulez: Couleur unie)
-Référence(par exemple vous pouvez mettre si vous voulez: NI122O0CE-Q11)
-Taille du mannequin(par exemple vous pouvez mettre si vous voulez: Notre mannequin mesure 187 cm et porte une taille L)
-Coupe(par exemple vous pouvez mettre si vous voulez: Normale)
-Longueur(par exemple vous pouvez mettre si vous voulez: Normale)
-Longueur des manches(par exemple vous pouvez mettre si vous voulez: Manches courtes)
-Longueur totale(par exemple vous pouvez mettre si vous voulez: 73 cm en taille L)
Dans le champ "Quantité" vous insérer le nombre de t-shirts du meme modèle que vous souhaitez vendre.

Dans le champ "Type de produit" vous insérer "t-shirt".

Dans le champ "Condition" vous insérer "neuf" si elle est neuf sinon insérer "occasion".

Dans le champ "Coordonnées de contact" vous insérer vos coordonnées par lesquelles le client peut vous contacter.

Dans le champ "Coordonnées du produit" vous insérer le lieu exact ou le client peut récupérer le t-shirt.

L'exemple que nous prenons est un pantalon en Jersey, mais il existe beaucoup d'autres matieres comme le Cachemire, Cachemire (100%), Coton, Coton (100%), Crocheté, Cuir, Côtelé, Denim, Dentelle, Flanelle chaude, Hardshell, Imitation cuir, Jersey, Laine, Laine (100%), Lin, Lin (100%), Lyocell (100%), Mesh, Paillettes, Polaire, Polyester, Satin, Softshell, Soie, Strié, Sweat, Textile, Tressé, Tricots, Viscose et Viscose (100%).
