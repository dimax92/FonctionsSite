Comment vendre ses chaussures

Une fois connecte a votre compte.

Pour vendre ces chaussures sur machatvente.com il faut d'abord cliquer sur "Déposer une annonce".

Dans le champ "Nom du produit" vous insérer le nom du modèle de la paire de chaussures que vous possédez.

Dans le champ "Marque du produit" vous insérer la marque de vos chaussures.

Cliquez sur "Choisir une vidéo" et ensuite choisissez la vidéo de présentation de vos chaussures, nous vous conseillons de prendre votre video en plaine lumière pour qu'ont vois bien en détail votre paire de chaussures et de montrer la paire de chaussures sous tous les angles possibles pour que le client potentiel est une vue détaillé de votre paire de chaussure. Filmez aussi l'intérieur de la chaussure car si l'intérieur de la chaussure comporte des problèmes, le confort du client pourra en être affecte quand il portera la paire de chaussure. Le poids de la video ne doit pas depasser 100mb, la video doit être courte.

Dans le champ "Prix du produit" vous insérer le prix que vous voulez pour votre paire de chaussures.

Dans le champ "Devise" vous insérer la devise que vous voulez(euros, dollars, bitcoin, …).

Dans le champ "Description du produit" vous insérer une présentation générale de vos chaussures.

Pour donner des détails sur vos chaussures cliquer sur le "+" a droite de "Details du produit".
Les détails que je vous conseille d'indiquer sont:
-Couleur(par exemple: noire, gris et noir, ...), vous pouvez mettre plusieurs couleurs si vous avez des paires de chaussures du meme modèle mais pas de la meme couleur.
-Taille(par exemple: 41, 44, ...), vous pouvez mettre plusieurs tailles si vous avez des paires de chaussures du meme modèle mais pas de la meme taille.
-Dessus / Tige(par exemple: Cuir, Textile, ...)
-Doublure(par exemple: Cuir, Textile, ...)
-Semelle de propreté(par exemple: Cuir, Textile, ...)
-Semelle d'usure(par exemple: Matière synthétique, ...)
-Épaisseur de la doublure(par exemple: Doublure protégeant du froid, ...)
-Conseils d'entretien(par exemple: Appliquez un imperméabilisant avant la première utilisation, ...)
-Bout de la chaussure(par exemple: Rond, ...)
-Forme du talon(par exemple: Plat, ...)
-Fermeture(par exemple: Laçage, ...)
-Motif / Couleur(par exemple: Couleur unie, ...)
-Informations additionnelles(par exemple: Découpes, ...)
-Référence(par exemple: NI112O0BT-Q12, ...)

Dans le champ "Quantité" vous insérer le nombre de paires de chaussures que vous souhaitez vendre.

Dans le champ "Type de produit" vous insérer "chaussures".

Dans le champ "Condition" vous insérer "neuf" si elles sont neuves sinon insérer "occasion".

Dans le champ "Coordonnées de contact" vous insérer vos coordonnées par lesquelles le client peut vous contacter.

Dans le champ "Coordonnées du produit" vous insérer le lieu exact ou le client peut récupérer la paire de chaussures.

Prenez bien soin de les nettoyer car des chaussures sales meme si le modèle peut intéresser le client, ne donnent pas envie de les achetés.
